## Sales Summary Dashboard

This repository contains a simple, interactive Streamlit application designed to help non-technical users easily explore basic sales data. 

The app is built using Python and utilizes a hardcoded Pandas DataFrame containing dummy sales records for various products across categories like Electronics and Furniture. 

Users can interact with the dashboard using a built-in sidebar dropdown menu to filter the dataset by a specific product category. 

Once a category is selected, the main view instantly updates to display both the filtered tabular data and a dynamic line chart visualizing those specific sales figures. 

Overall, this project demonstrates fundamental Streamlit capabilities, including layout structuring, data filtering, and basic chart rendering.
